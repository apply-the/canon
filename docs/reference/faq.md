## Integration & Tools

### Is Canon dependent on Boundline?
**No.** Canon is useful on its own as a semantic governance runtime. While they are complementary, you can use Canon to govern knowledge for any downstream delivery system or manual process.

### How do I integrate Canon with other tools?
Read [Boundline Integration](/guide/core-concepts), then check the source repository for the [Governance Adapter Integration](https://github.com/apply-the/canon/blob/main/tech-docs/integration/governance-adapter.md) guide.
